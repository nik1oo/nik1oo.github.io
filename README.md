
# Directory Structure

 - `generate` -- The site generator.
 - `source` -- Input.
	 - `content` -- Inpu Markdown.
	 - `scripts` -- Input Typescript.
	 - `templates` -- Input HTML.
 - `docs` -- Output.
 - `temp` -- Temporary files produced by the generator.
